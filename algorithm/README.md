# Algorithm 算法学习

个人算法学习记录项目，基于 [LEETCODE_GUIDE.md](./LEETCODE_GUIDE.md) 制定的系统刷题计划，按专题分阶段推进。

---

## 目录结构

```
leetcode/
├── structure/          公共数据结构
│   ├── ListNode        链表节点
│   ├── TreeNode        二叉树节点
│   └── Node            图节点
├── phase1/             第1-2周 前置：基础数据结构（手写实现 + 设计题）
├── topic1/             第3周  专题1: 数组 & 哈希表
├── topic2/             第4周  专题2: 双指针 & 滑动窗口
├── topic3/             第5周  专题3: 链表
├── topic4/             第6周  专题4: 栈 & 队列
├── topic5/             第7周  专题5: 二叉树
├── topic6/             第8周  专题6: 图 & BFS/DFS
├── topic7/             第9周  专题7: 二分搜索
├── topic8/             第10周 专题8: 动态规划
├── topic9/             第9周  专题9: 回溯（穿插）
└── phase3/             第11-14周 综合强化
```

每个专题目录下包含：
- `WeekPlan.java` — 本周每日计划 + 间隔复习追踪表（第2天/第7天/第30天）
- 各题目骨架文件（`Q{题号}{题名}.java`）及对应测试文件

## 每日刷题节奏

```
复习昨日题目（20分钟）→ 新题（60-90分钟）→ 总结归纳（10分钟）

周一至周五：按专题刷新题（1-2题/天）
周六：复习本周所有题，整理笔记
周日：参加 LeetCode 周赛 或综合练习
```

**间隔复习（Spaced Repetition）**

| 节点 | 操作 |
|------|------|
| 第1天 | 做题 |
| 第2天 | 不看答案重做 |
| 第7天 | 再做一次 |
| 第30天 | 最终巩固 |

每个 `WeekPlan.java` 中包含本专题的完整间隔复习追踪表。
